# Fraud Detection Classifier

This project builds a Machine Learning model to predict whether there is a Fraud or not.

## Dataset

- Dataset used: Credit Card Fraud Detection

Download it from : 
[File Link](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)

After downloading, place the files inside "dataset" folder.

## Task

- Handeled imbalanced dataset
- RandomForestClassifier for prediction
- GridSearchCV for hyperparameter tuning


## Library Used
- pandas
- scikit-learn
